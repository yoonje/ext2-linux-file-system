# EXT2-Linux-File-System
EXT2 File System emulator. It works like Linux does.<br>

# Run

```sh
$ make
```

```sh
$ ./shell
```

```sh
$ [/]# format
```

```sh
$ [/]# mount
```

After this commands you can use several command like Linux(touch, mkdir, mv, ls, cd...)
